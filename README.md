# Beneath the Surface

This project was made for a school hackathon with the theme “Beneath the Surface.” The idea of the website is that people and things can have deeper meanings hidden underneath what we first see. I used an ocean theme because the ocean also has many hidden layers below the surface.

The website was made using HTML, CSS, and a little bit of JavaScript. I added animations, floating bubbles, and scroll effects to make the website feel more alive and interactive.

One challenge was making the animations work smoothly together, but after changing some values and testing the code, it worked better.

Overall, this project was meant to be simple but meaningful while matching the hackathon theme.
